Differnce Between Local Storage and Session Storage?

SessionStorage and LocalStorage almost do the same work, but there are some differences between them,
localStorage uses for the long process because it's stores data and keeps it for the next use but when you use sessionStorage it will expire when your window/tab will be closed.
In short: you will use localStorage when you keep the data unit you delete or user delete and you will use sessionStorage for temporary use.
sessionStorage data will be unique for different tabs but localStorage Data can't be unique.
sessionStorage data shared between HTTP and HTTPS but localStorage data won't share.
Example: when you mute youtube video it will remain for next time untill you unmute, it's use localStorage.
